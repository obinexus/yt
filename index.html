Detach Mode Guidelines

  • 0-3: Safe to detach, system stable
  • 3-6: Warning zone, detach with caution
  • 6-9: Critical errors, fix before detaching
  • 9-12: System panic, immediate intervention required

Use --no-panic flag to extend error tracing beyond 12 for debugging.
